# Register Application
在GitHub上注册新应用，链接：https://github.com/settings/applications/new
注册完成后，会有Client ID 和Client Secret


# 配置主题_config.yml文件
最新版的NexT主题已经集成了gitalk的功能，所以注册完应用后直接修改配置文件就可以了。
