nmon

检查安装环境

1、查看版本内核

2、查看系统版本

下载

根据系统版本下载对应版本的nmon：
http://nmon.sourceforge.net/pmwiki.php?n=Site.Download
CentOS7安装nmon16e_mpginc.tar.gz
nmon analyze下载：
https://www.ibm.com/developerworks/community/wikis/home?lang=en#!/wiki/Power+Systems/page/nmon_analyser

安装

  • 安装位置/opt/nmon

  • 在opt目录下创建nmon文件夹

  • 上传nmon16e_mpginc.tar.gz到nmon目录

  • 解压

  • 赋权限

  • 启动

配置环境变量

  • 修改启动文件名称

  • 添加到配置文件中

PATH=$PATH:/opt/nmon/nmon $PATH:后为命令的路径
export PATH
保存后退出

  • 使配置文件立即生效

数据采集

命令说明
-s1每隔n秒抽样一次，这里为1秒
-c30取出多少个抽样数量，这里为30，即监控=1*30/60=0.5分钟
-f按标准格式输出文件名称：_YYMMDD_HHMM.nmon
-m指定监控文件的存放目录，-m后跟指定目录
-t输出最耗资源的进程
-r test监控记录的标题

生成图形化报表

  • 将.nmon文件转化为.csv文件

  • 将csv文件下载到本地

  • 在本地解压nmon_analyser工具，打开nmon _analyser_v60.xlsm
  • 在Analyser页签，点击Analyze nmon data加载csv文件

    结束nmon运行

    ps -ef|grep nmon
    kill -9 pid
